### Alternative zur Input-Klasse ###

Wem das zu unschön ist, eine eigene Klasse für den Input anzulegen, darf sich auch der Klasse java. io. Console bedienen. Beachtet bitte, dass es diese erst ab Java 6 in dieser Form gibt und sich Details dazu in Java 7 nochmal geändert haben.
