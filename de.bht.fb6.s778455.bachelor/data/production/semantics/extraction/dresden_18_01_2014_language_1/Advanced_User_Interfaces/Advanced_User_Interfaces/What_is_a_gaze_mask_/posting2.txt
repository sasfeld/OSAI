Well to me it is a broad description for techniques such as attentive/gaze-contingent displays. In the case of gaze-contingent displays the area which is inspected by the eye is treated specially or the rest of the display is blured out (see page 38). Hence, a mask is applied to the display which is mainly influenced by the eye.
