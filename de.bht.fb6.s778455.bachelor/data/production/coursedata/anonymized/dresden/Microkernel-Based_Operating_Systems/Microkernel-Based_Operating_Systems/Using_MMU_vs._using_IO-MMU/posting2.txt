CONTENT:
The MMU and IOMMU sit (conceptually) between different parts of the computer. Imagine MMUs sitting between each CPU and RAM. An IOMMU sits between devices and RAM. So if a CPU accesses memory using a virtual address the MMU translates this. If a device accesses memory, the IOMMU translates this.
