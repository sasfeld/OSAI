- Was sind die Hauptteile einer Java-Klasse? Attribute, Konstruktoren, Methoden. Eine Java Klasse kann auch innere Klassen beinhalten (siehe LE über Ausnahmen)

- Was heißt eine Methode überladen? Was heißt eine Methode überschreiben?
Eine Methode ist überladen, wenn sie mit dem gleichen Namen aber verschiedenen Parameterlisten (gleiche Signatur) mehrmals in einer Klasse vorkommt. Beispiel: Klasse StringBuilde Methode append.
Eine Methode ist überschrieben, wenn sie exact mit dem gleichen Kopf in eine Unterklasse neu implementiert ist. Beispiel: die Methoden toString oder equals der Klasse Object werden in vielen Unterklassen überschrieben.

- In den folgenden Code-Zeilen:
Triangle t1 = new Triangle(3,4,5);
System.out.println(t1.toString());
woher kommt die Methode toString()?
Da die Methode toString in der Klasse Triangle nicht existiert, muss sie in einer Oberklasse der Klasse Triangle deklariert worden sein. In diesem Fall ist sie in der Klasse Object deklariert.
